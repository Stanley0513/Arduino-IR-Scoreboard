# Arduino-IR-Scoreboard

這是一個以 **C 語言** 開發的硬體控制專案，主要實作一套「電子計分板」系統。透過 Arduino Uno R3 微控制器結合紅外線感測器，達成實體訊號與程式邏輯的互動。

## 🛠 技術棧 (Tech Stack)
* **語言:** C / C++
* **硬體:** Arduino Uno R3, IR Remote & Sensor, LCD (16x2)
* **核心概念:** 外部中斷處理、硬體訊號解碼、PWM 顯示控制

## 🌟 核心功能 (Features)
* **多種賽制支援:** 內建排球 (25分制)、桌球 (11分制) 與羽球 (21分制) 的計分規則。
* **紅外線即時遙控:** 透過 IR 遙控器即時控制分數增減、勝利局數增減以及雙方換場。
* **防呆與狀態管理:** 實作按鈕去抖動與狀態延遲機制，確保訊號接收的穩定性。
* **LCD 即時顯示:** 將即時比分與勝利提示動態輸出至 16x2 LCD 螢幕。

## 📂 專案結構
* `期末程式原始碼.ino`: 包含主迴圈、IR 訊號處理與 LCD 顯示邏輯的核心原始碼。
* `期末報告.pdf`: 包含系統架構圖、硬體接線圖與詳細功能說明。
